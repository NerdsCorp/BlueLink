# Multi Arduino Controller (Auth + React)


## Setup


### Backend
```bash
cd backend
python -m venv venv
source venv/bin/activate # or venv\\Scripts\\activate
pip install -r requirements.txt
python backend_init.py
python app.py
```


### Frontend
```bash
cd frontend
npm install
npm run dev
```


### Access
Open http://localhost:5173 (dev) or http://localhost:8000 (if served).
